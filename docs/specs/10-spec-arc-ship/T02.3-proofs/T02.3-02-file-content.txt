# Cross-Plugin Contract

What Arc reads from Temper and when. Arc never writes to Temper artifacts — all access is read-only with graceful degradation.

## Claude-Workflow Artifacts Read by Arc

## Graceful Degradation

All reads from external plugins are conditional:
- **Temper artifacts**: If the file exists, read it for context. If not, proceed without it — Temper may not be installed. Never error or warn on missing Temper artifacts unless explicitly checking for engineering readiness.
- **Claude-Workflow artifacts**: If validation or proof artifacts exist, read them for decision support. If not, proceed without them — the workflow may not have executed yet or in earlier plugin versions.

## Directionality

Arc → Temper and Arc → Claude-Workflow access are strictly **read-only**. Arc never creates, modifies, or deletes artifacts managed by other plugins. This ensures all plugins can operate independently and compose without conflict.
