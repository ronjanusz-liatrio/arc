This was rewritten to "updates ROADMAP and BACKLOG (does not write to CLAUDE.md — run /arc-sync for the product-context block)" to reflect the new factual reality. The substring "product-context" remains in the frontmatter description; this is outside the numbered execution steps and is permitted by the spec proof grep, which only forbids the substring inside numbered steps.

The option's user-facing intent is unchanged — invoking it still tells the user to run /arc-sync.
